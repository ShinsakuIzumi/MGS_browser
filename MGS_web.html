How to Use Simulator

  1. Select or import a reference image using the “Reference Image” menu and button.
  2. Set the number and speed of agents and the contrast of the image of the resulting formation using the corresponding menus.
  3. Perform the simulation of the mass game by clicking the “Simulation” button. The progress of the simulation is shown on the “Simulation/Animation Progress” bar.
  4. The resulting formation can be viewed in the “Formation” window using the “Time” slider and the “Animation” button.
